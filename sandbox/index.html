Sandbox

The sandbox allows you to run LPS programs in the browser using the bundled-for-browser version of lps.js. Certain modules provided by lps.js may not be available in the browser, such as the p2p module.

To get you started, you can choose to load the provided example LPS programs and click on "Run Program" to run the program. The program's execution timeline will then be shown below.
